Site Blocker
 

Some websites, such as CNN, load content from external web sites such as Facebook or Twitter. This can cause your corporate proxy or web filter to raise messages saying that the content is blocked. In many cases these messages are annoying. This plug-in prevents these websites from loading content from the websites you identify thus preventing those annoying messages. This plug-in does not enable to get around your corporate proxy or web filter.

If you have questions, feedback or enhancement requests please direct them to zechariahs+siteblocker@gmail.com.

Options

List all of the sites you want to block into the field below. One site per line. The list can be as specific as http://facebook.com or as broad as facebook. Changes take effect immediately.
